##### (기존 기록 방식)
- plus : 기간 동안 도움이 되었거나 즐거웠던 일들  
- minus : 기간 동안 해결해야 될 이슈라고 생각하는 것이나 나쁘거나 슬펐던 일들  
- interest / soso / try  
  - interest : 기간동안 흥미로운 것
  - soso : 좋고 나쁨의 감성에 포함되지 못하는 것들
  - try : 차주에 시도해 봤으면 하는 것
